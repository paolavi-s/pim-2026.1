## Sobre o Projeto

A **Gateferia Nekoffee** é uma cafeteria temática fictícia que combina a experiência gastronômica com interação terapêutica com gatos resgatados. O sistema desenvolvido visa promover o bem-estar emocional de pessoas com ansiedade, depressão e solidão, ao mesmo tempo em que incentiva a adoção responsável de animais.

O projeto foi desenvolvido com foco em **qualidade, usabilidade, acessibilidade e impacto social**.

---

## Funcionalidades Principais

- **Agendamento online** de visitas à gateferia
- **Processo de adoção digital** completo
- **Dashboard de métricas** e indicadores de impacto terapêutico
- **Recomendações de adoção** (preparado para Machine Learning)
- **Modo Escuro** e **Alto Contraste**
- **Integração com VLibras** (acessibilidade para surdos)
- **Interface totalmente responsiva** (Mobile-First)
