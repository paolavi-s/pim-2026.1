Desenvolvimento de API com C#

 - A linguagem C# foi utilizada no projeto, juntamente com os FrameWorks asp.net e Entity FrameWork para realizar a comunicação dentre a página HTML em si e o banco de dados, onde estão armazenados dados de Clientes, Dados bancários e outras informações pertinentes para o funcionamento do site.

 - Dessa forma, a API é responsável por:
	 - Comunicar a solicitação do usuário (referente a qual item ele deseja) para o banco de dados.
	 - Confirmar status de Entidades, e atualizar no sistema interno, caso alguma informação esteja divergente
	 - Realizar inclusão, atualização e exclusão de entidades da tabela de dados, além de ser responsável por carregar os dados internos (como informações bancárias) para o front-end.
